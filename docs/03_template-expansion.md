# みつもり主任 階層間・計算式テンプレート動的展開仕様

> **本ドキュメントはメインドキュメント（README.md）から参照されます。**
> 部屋別・部位別のテンプレート展開処理を行う場合に参照してください。

---

## 1. 概要

部屋や部位ごとに共通の計算式テンプレートを使い回すための「階層間文字列置換ロジック」を定義する。
親階層に設定された「識別子」を、子階層の計算式内の「プレースホルダー」に動的に展開することで、計算式管理を最適化し、AIによる推論エラーを防止する。

---

## 2. 構成要素の定義

### 2.1. 親階層（識別子保持レコード）

- **対象**: 階層フラグが `1` または `-1`（内訳開始ヘッダ）のレコード
- **役割**: 配下の子階層に適用する「置換用識別子」を保持
- **格納場所**: `計算式` 列に文字列として格納（例: `[部屋1]`, `[1階]`）

### 2.2. 子階層（テンプレート保持レコード）

- **対象**: 親階層の直下に配置される階層 `2` 等の明細レコード
- **役割**: 変数の添え字部分を空にした「汎用計算式テンプレート」を保持
- **プレースホルダー**: `[]`（**中身が空の半角角括弧**）

テンプレート例:
```
=部屋内法長[]*天井高[]-(腰壁高以上開口面積[]+巾木開口面積[])
```

---

## 3. 動的置換アルゴリズム

CSVを **行単位で上から順に走査** し、以下のルールで置換を実行する。

### ルール1: 識別子の取得と記憶

親階層（階層 `1` または `-1`）を読み込んだ際、`計算式` 列に値があれば「カレント識別子」として保持する。親階層が変わるたびに上書き更新。空欄の場合はクリアする。

### ルール2: テンプレートの展開（置換）

子階層を読み込んだ際、`計算式` 列に `[]` が存在し、かつ「カレント識別子」が保持されている場合、**式内のすべての `[]` を「カレント識別子」で単純文字列置換** する。

### ルール3: 例外規定（安全装置）

- `[105]` や `[3]` のように括弧内に既に文字・数値が入っている場合は **置換対象外**。完全に空の `[]` のみを置換
- カレント識別子が未設定、または子階層の計算式が空欄の場合は何も処理しない

---

## 4. 処理例

### 変換前（テンプレート状態）

| 階層 | 項目名 | 計算式（格納値） | エンジン内部状態 |
|:---|:---|:---|:---|
| `1` | 寝室 | `[部屋1]` | カレント識別子 = `[部屋1]` |
| `-1` | 寝室 | | カレント識別子 = `[部屋1]`（維持） |
| `2` | 廻縁 | `=部屋内法長[]` | 置換発動 |
| `2` | 壁仕上 | `=部屋内法長[]*天井高[]-(腰壁高以上開口面積[]+巾木開口面積[])` | 置換発動 |

### 変換後（完成データ）

| 階層 | 項目名 | 計算式（置換後） |
|:---|:---|:---|
| `1` | 寝室 | `[部屋1]` |
| `-1` | 寝室 | |
| `2` | 廻縁 | **`=部屋内法長[部屋1]`** |
| `2` | 壁仕上 | **`=部屋内法長[部屋1]*天井高[部屋1]-(腰壁高以上開口面積[部屋1]+巾木開口面積[部屋1])`** |

この変換により、計算エンジンは `部屋1` という変数を評価し、基本数量シートに定義された `天井面積[1]` などの配列変数の値を正確に参照可能となる。

### 処理例2（店舗の場合）

元仕様書にある別の例:

#### 変換前

| 階層 | 項目名 | 計算式（格納値） | エンジン内部状態 |
|:---|:---|:---|:---|
| `1` | 1:2階-<店舗> | `[部屋1]` | カレント識別子 = `[部屋1]` |
| `-1` | 1:2階-<店舗> | | カレント識別子 = `[部屋1]`（維持） |
| `2` | 天井仕上 C級ｸﾛｽ | `=天井面積[]` | 置換発動 → `[]` を `[部屋1]` に |
| `2` | 壁仕上 C級ｸﾛｽ | `=部屋内法長[]*天井高[]` | 置換発動 → `[]` を `[部屋1]` に |

#### 変換後

| 階層 | 項目名 | 計算式（置換後） |
|:---|:---|:---|
| `1` | 1:2階-<店舗> | `[部屋1]` |
| `-1` | 1:2階-<店舗> | |
| `2` | 天井仕上 C級ｸﾛｽ | **`=天井面積[部屋1]`** |
| `2` | 壁仕上 C級ｸﾛｽ | **`=部屋内法長[部屋1]*天井高[部屋1]`** |
