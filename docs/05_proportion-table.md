## 2. 設備工事

| :--- | :--- | :--- | :--- | :--- | :--- |
| **給排水衛生設備工事** | 給水配管 | 水栓箇所数×配管延長（m） | 配管材種（架橋PE/銅管等） | 長さ比例＋個数 | 水廻りの集約度で配管長が変動 |
| | 排水配管 | 排水箇所数×配管延長（m） | 配管材種・勾配 | 長さ比例＋個数 | 2階に水廻りがあると配管増 |
| | 給湯配管 | 給湯箇所数×配管延長（m） | 給湯方式（エコキュート/ガス等） | 長さ比例＋個数 | ヘッダー方式か先分岐方式かで変動 |

| | 配線・配管 | 回路数×配線延長（m） | 階数・間取り | 長さ比例＋回路数 | 延床面積・部屋数に概ね比例 |
| | コンセント・スイッチ | 設置数（箇所） | 仕様（標準/USB付/調光等） | 個数比例 | 部屋数×2～3箇所が目安 |
| | 照明器具 | 設置数（箇所） | 器具グレード | 個数×単価 | ダウンライト数や間接照明で大きく変動 |
| | TV/LAN配線 | 端末数（箇所） | 配線方式（有線/無線） | 個数比例 | LAN配線省略の場合もあり |
| | インターホン・火災報知器 | 数量（通常1式） | 機能（モニター付/録画等） | 固定＋グレード | 火災報知器は居室数に比例 |
| **空調換気設備工事** | エアコン | 設置台数 | 能力（畳数）・グレード | 台数×単価 | 全館空調の場合は1式で高額 |
| | 24時間換気システム | 換気対象面積（延床㎡） | 方式（1種/3種） | 面積比例＋方式 | 第1種熱交換は第3種の3～5倍 |
| | 浴室換気乾燥機 | 数量（通常1箇所） | グレード | 固定＋グレード | |
| **ガス設備工事** | ガス配管・機器接続 | ガス機器数（箇所）×配管延長（m） | 都市ガス/LPG | 長さ比例＋個数 | オール電化の場合は不要 |
