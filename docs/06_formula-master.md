# 【知識ベース】見積計算式マスターデータ
以下の変数は、システムで自動計算可能な基本数量と、それに紐づく計算式のリストです。

## 1F床面積
- 分類: 仮設
- 単位: ㎡
- 計算式: `=1F床面積`
- 説明: １階床面積の㎡数を拾った計算式です。

## ﾒｯｼｭｼｰﾄ
- 分類: 仮設
- 単位: ㎡
- 計算式: `=COND(1F外周領域周長>2F外周領域周長,1F外周領域周長,2F外周領域周長)*地上階数*3*1.2`
- 説明: 1階または2階の外周の長さに階数を掛けて、階高を3ｍと設定しているので、3を掛けます。さらに1.2の数値は足場の周りにメッシュシートを掛けるので係数の1.2を掛けた、㎡面積を拾った計算式です。

## 延床面積(㎡)
- 分類: 仮設
- 単位: ㎡
- 計算式: `=延床面積`
- 説明: 延床面積の㎡数を拾った計算式です。

## 延床面積(坪)
- 分類: 仮設
- 単位: 坪
- 計算式: `=延床面積*0.3025`
- 説明: 延床面積の㎡数を坪に直した計算式です。

## 昇降設備階段
- 分類: 仮設
- 単位: ｍ
- 計算式: `=地上階数*3`
- 説明: 足場と足場に掛ける階段のことで、階数に対して、係数の3を掛けた計算式です。

## 屋根足場（６寸以上）
- 分類: 仮設
- 単位: ㎡
- 計算式: `=(COND(1F屋根勾配>=6,1F屋根面積,0)+COND(2F屋根勾配>=6,2F屋根面積,0)+COND(3F屋根勾配>=6,3F屋根面積,0)+COND(4F屋根勾配>=6,4F屋根面積,0))*ロス`
- 説明: 6寸以上勾配以上の一般屋根の㎡数を拾った計算式です。

## 仮設延面積
- 分類: 仮設
- 単位: ㎡
- 計算式: `=(延床面積+吹抜面積)*ロス`
- 説明: 延床面積と、吹き抜け面積を足した、㎡数を拾った計算式です。

## 仮設建築面積
- 分類: 仮設
- 単位: ㎡
- 計算式: `=(1F床面積+ﾎﾟｰﾁ平面面積+土間ｺﾝ平面面積)*ロス`
- 説明: 1階の床面積とポーチ床面積と土間コンクリートの平面面積を足した㎡数を拾った計算式です。

## 内部足場
- 分類: 仮設
- 単位: ㎡
- 計算式: `=吹抜面積*ロス`
- 説明: 吹き抜けの㎡数を拾った計算式です。

## 基礎面積（概算）
- 分類: 基礎
- 単位: ㎡
- 計算式: `=1F床面積*ロス`
- 説明: １階床面積の㎡面積を拾った計算式です。１階床面積を概算の基礎面積としてみています。
