# 木造住宅 積算数量算出リファレンス
**準拠：別添一の一 木造建物調査積算要領〔軸組工法〕**
このリファレンスは見積データに計算式を組み込む際、他の情報で解決できない場合に最終的な判断を行うために使用するルールです。
---

## 1. 入力が必要な建築諸元（調査項目）

### 最優先（ほぼ全工種で使用）

| 諸元 | 単位 | 用途 | 根拠 |
|------|------|------|------|
| 1階床面積 | ㎡ | 基礎率の面積区分判定、基礎長算出、樋工事費 | 図面作成基準第6 |
| 2階床面積 | ㎡ | 延床面積の算出 | 同上 |
| 延床面積 | ㎡ | 仮設・軸部・塗装工事費 | =1階+2階床面積 |
| 用途区分 | 窶鐀 | 基礎率・木材材積率の選択 | 数量積算基準第1 |

### 高優先（個別工種で必須）

| 諸元 | 単位 | 用途 | 根拠 |
|------|------|------|------|
| 1階外壁周長 | m | 布基礎仕上げ外周長、外壁面積 | 第10条（柱中心間） |
| 2階外壁周長 | m | 外壁面積 | 同上 |
| 1階壁高 | m | 外壁面積 | 第10条（外壁下端～軒桁上端） |
| 2階壁高 | m | 外壁面積 | 同上 |
| 外壁面数（1階） | 面 | 仮設の建物形状補正率 | 第6条（出幅45cm以内の出窓除く） |
| 柱径 | mm | 木材材積率の選択 | 第8条（最多使用柱） |
| 柱長（1階/2階） | m | 木材材積率の選択 | 第8条 |
| 屋根勾配 | /10 | 屋根施工面積の伸び率 | 第9条 |
| 屋根伏面積 | ㎡ | 屋根施工面積 | 屋根伏図 |

### 個別調査が必要な項目

各室の仕上材種、開口部の種類・寸法・数量、造作の種類・数量、建築設備の種類・数量など（第11条～第19条による現地調査）

## 2. 面積区分（共通）

多くの諸率が以下の7区分に対応する。先に判定しておくと全工種で使い回せる。

| 区分 | Ⅰ | Ⅱ | Ⅲ | Ⅳ | Ⅴ | Ⅵ | Ⅶ |
|------|------|------|------|------|------|------|------|
| 範囲 | 50㎡未満 | 50以上70未満 | 70以上100未満 | 100以上130未満 | 130以上180未満 | 180以上250未満 | 250㎡以上 |

- **仮設・軸部・塗装**→延床面積で判定
- **基礎**→1階床面積で判定（べた基礎立上は底盤施工面積で判定）

## 3. 工種別の数量算出方法

### 3-1. 仮設工事費（第27条／数量積算基準第3）

```
仮設工事面積 = 延床面積 × 規模補正率 × 建物形状補正率
工事費 = 仮設工事面積 × 単価
```

#### 規模補正率（延床面積の区分）

| 区分 | Ⅰ | Ⅱ | Ⅲ | Ⅳ | Ⅴ | Ⅵ | Ⅶ |
|------|------|------|------|------|------|------|------|
| 補正率 | 1.15 | 1.10 | 1.05 | 1.00 | 0.90 | 0.85 | 0.75 |

#### 建物形状補正率（1階外壁面数）

| 6面以下 | 7面以上10面以下 | 11面以上 |
|--------|-------------|---------|
| 1.00 | 1.10 | 1.20 |

**注**：築年次が異なる建物が接合している場合は一体として延床面積を算出し対応する率を用いる（数量積算基準第3三）。

---

### 3-2. 基礎工事費（第28条／数量積算基準第4）

#### ① 布コンクリート等基礎

```
布基礎長 = 1階床面積 × 基礎率
工事費 = 布基礎長 × 単価
```

#### ② 布基礎仕上げ

```
基礎外周長 = 1階外壁周長
工事費 = 基礎外周長 × 単価
```

#### ③ 束石

```
束石数量 = 1階床面積 × 束石基礎率 ……専用住宅の場合
束石数量 = 束石施工面積 × 束石基礎率 ……専用住宅以外
工事費 = 束石数量 × 単価
```

#### ④ べた基礎

```
底盤工事費 = 1階の底盤施工面積 × 単価
立上工事費 = (底盤施工面積 × 基礎率) × 単価
べた基礎工事費 = 底盤工事費 + 立上工事費
```

#### ⑤ べた基礎仕上げ

```
基礎外周長 = 底盤部分外周長（柱中心間測定値）
工事費 = 基礎外周長 × 単価
```

#### ⑥ 独立基礎・玉石基礎

```
工事費 = 調査数量 × 単価
```

#### ⑦ 土間コンクリート

```
工事費 = 施工面積（調査値） × 単価
```

#### ⑧ 床下防湿コンクリート

```
工事費 = 施工面積（調査値） × 単価
```

#### 基礎率表（1階床面積1㎡当たり）

**布基礎（m/㎡）**

| 用途＼面積区分 | Ⅰ | Ⅱ | Ⅲ | Ⅳ | Ⅴ | Ⅵ | Ⅶ |
|------------|------|------|------|------|------|------|------|
| 専用住宅 | 1.16 | 1.10 | 1.04 | 0.96 | 0.88 | 0.85 | 0.79 |
| 共同住宅 | 1.18 | 1.13 | 1.07 | 1.00 | 0.92 | 0.89 | 0.84 |
| 店舗・事務所 | 0.95 | 0.89 | 0.84 | 0.76 | 0.69 | 0.65 | 0.59 |
| 工場・倉庫 | 0.67 | 0.62 | 0.56 | 0.49 | 0.41 | 0.34 | 0.28 |

**束石（個/㎡）**　※専用住宅の率を使用

| 区分 | Ⅰ | Ⅱ | Ⅲ | Ⅳ | Ⅴ | Ⅵ | Ⅶ |
|------|------|------|------|------|------|------|------|
| 束石 | 0.43 | 0.44 | 0.45 | 0.47 | 0.48 | 0.51 | 0.55 |

**補足ルール**
- 1階が2以上の用途に区分→1階床面積に対応した面積区分で用途ごとの基礎率を適用（数量積算基準第4-2）
- 一つの用途で基礎種類が異なる→種類ごとに施工面積を算出し面積区分の基礎率を適用（同第4-3）
- 基礎率の使用が困難な形状・間仕切り→別途個別に基礎長を算出（同第4-4）
- 築年次が異なる接合建物→一体として1階床面積を算出し対応する基礎率を使用（同第4-5）

```
軸部工事費 = (木材材積量 × 木材単価) + (延床面積 × 労務費単価)
```

### 3-4. 屋根工事費（第30条／数量積算基準第6）

```
屋根施工面積 = 屋根伏面積 × 屋根勾配伸び率
工事費 = 施工面積 × 単価（仕上材種別の合計）
```

#### 屋根勾配伸び率 = √(1 + (勾配/10)2)

| 勾配 | 1/10 | 1.5/10 | 2/10 | 2.5/10 | 3/10 | 3.5/10 | 4/10 | 4.5/10 | 5/10 | 5.5/10 |
|------|------|--------|------|--------|------|--------|------|--------|------|--------|
| 伸び率 | 1.005 | 1.011 | 1.020 | 1.031 | 1.044 | 1.059 | 1.077 | 1.097 | 1.118 | 1.141 |

---

### 3-5. 外壁工事費（第31条／数量積算基準第7）

```
外壁施工面積 = 外壁面積（開口部含む）竏鈀 外壁開口部面積
工事費 = 施工面積 × 単価（仕上材種別の合計）
```

- 外壁面積 = 外壁周長 × 壁高（各階・各仕上材種別）
- 切妻の場合は妻面積を別途加算
- **1か所あたり0.50㎡以下の開口部は控除しない**
- 様式第2で算出

---

### 3-6. 内壁工事費（第32条／数量積算基準第8）

```
内壁施工面積 = 内壁面積（開口部含む）竏鈀 内壁開口部面積
工事費 = 施工面積 × 単価（仕上材種別の合計）
```

- **1か所あたり0.50㎡以下の開口部は控除しない**
- **階段室の内壁施工面積は10.80㎡**（直階段・廻り階段・折返し階段すべて共通、1階床～2階床まで）
- 様式第3で算出

---

### 3-7. 床工事費（第33条／数量積算基準第9）

```
床仕上材種：工事費 = 施工面積 × 単価（仕上材種別の合計）
畳敷き　　：工事費 = 帖数 × 単価（畳の材種別の合計）
```

- 様式第4で算出

---

### 3-8. 天井工事費（第34条／数量積算基準第10）

```
工事費 = 施工面積 × 単価（仕上材種別の合計）
```

- 様式第4で算出

---

### 3-9. 開口部〔金属製建具〕工事費（第35条／数量積算基準第11）

```
工事費 = 数量 × 単価（種類別の合計）
```

**サッシュ窓**（ルーバー・固定式除く）：以下4種類別に窓面積の合計を求める
- ア：雨戸無し面格子無し
- イ：雨戸無し面格子有り
- ウ：雨戸有り鏡板無し
- エ：雨戸有り鏡板有り

ルーバー・固定式は各々の窓面積合計。
玄関・勝手口・窓手摺り・出窓等は種類・材質・規格寸法別の箇所数。
様式第5で算出。

---

### 3-10. 開口部〔木製建具〕工事費（第36条／数量積算基準第12）

```
工事費 = 数量 × 単価（種類別の合計）
```

以下の種類別に建具枚数の合計を求める。
- ア：フラッシュ戸（戸ふすま含む）
- イ：ガラス戸、窓
- ウ：雨戸
- エ：障子
- オ：ふすま

格子戸・戸袋・出窓等は種類・材質・規格寸法別の建具枚数又は箇所数。
様式第6で算出。

---

### 3-11. 造作工事費（第37条）

```
工事費 = 数量 × 単価（種類別の合計）
```

第16条の調査数量を使用（床の間、書院、床脇、欄間、造付けタンス、階段、手摺、押入れ、造付け下駄箱、床下収納庫、掘りこたつ、霧除庇等）。
ただし軸部工事の木材材積量に含まれる構造部材は除く。

---

### 3-12. 樋工事費（第38条）

```
工事費 = 1階床面積 × 単価
```

---

### 3-13. 塗装工事費（第39条）

```
工事費 = 延床面積 × 単価
```

---

### 3-14. 建築設備工事費（第40条／数量積算基準第13）

#### ① 電気設備

| 項目 | 数量の求め方 |
|------|----------|
| 電灯 | 設置されている電灯数量 |
| スイッチ | 規格別数量（調査値） |
| コンセント | 規格別数量（調査値） |
| 分電盤 | 規格別数量（調査値） |
| 配管配線 | 電灯＋スイッチ＋コンセント＋分電盤の**合計数** |

#### ② ガス設備

- 都市ガス：各地域の工事費実態により算出
- プロパンガス：調整器等設置費 ＋（配管数量×単価）＋（ガス栓数量×単価）

#### ③ 給水・給湯設備

```
工事費 = 水栓工事費 + 建物内配管工事費 + 建物外配管工事費
```

| 項目 | 数量の求め方 |
|------|----------|
| 水栓工事費 | 水栓種類別数量 × 単価（洗面器具・水洗便器等に附随する水栓は**除く**） |
| 建物内配管工事費 | 水栓数量 × 単価（洗面器具・水洗便器等に附随する水栓を**含む**） |
| 建物外配管工事費 | 本管取付～計量器の工事費 ＋（計量器からの配管数量×単価） |

#### ④ 排水設備

```
工事費 = 建物内排水設備工事費 + 建物外排水設備工事費
```

| 項目 | 数量の求め方 |
|------|----------|
| 建物内 | 水栓数量 × 単価（洗面器具・水洗便器等に附随する水栓を**含む**） |
| 建物外 | （種類別配管数量×単価）＋（桝等の数量×単価）（調査値） |

#### ⑤ 衛生設備・⑥ 厨房設備・⑦ その他設備

```
工事費 = 数量 × 単価（種類別の合計）
```

いずれも第18条の調査数量を使用。

---

### 3-15. 建物附随工作物工事費（第41条）

```
工事費 = 数量 × 単価（種類別の合計）
```

第19条の調査数量（テラス、ベランダ等）を使用。

---

### 3-16. その他の工事費（第42条）

上記工種以外の工事がある場合、上記の方法に準じて算出。

---

## 4. 共通仮設費・諸経費（第43条～第44条）

```
直接工事費 = 3-1～3-16の各工事費の合計
共通仮設費 = 直接工事費 × 3%
純工事費 = 直接工事費 + 共通仮設費
諸経費 = 純工事費 × 諸経費率
推定再建築費 = 純工事費 + 諸経費
```

| 22超～24以下 | 27.9 | 140超～160以下 | 17.6 |

| 26超～28以下 | 26.9 | 180超～200以下 | 16.7 |

## 6. 計算の流れまとめ

```
[調査] 1階床面積、2階床面積、外壁周長、壁高、外壁面数、柱径、柱長、屋根勾配、屋根伏面積…
   │
   ├─ 延床面積 = 1階 + 2階
   ├─ 面積区分の判定（延床用 / 1階床用）
   │
   ├─[仮設] 延床 × 規模補正率 × 形状補正率
   ├─[基礎] 1階床 × 基礎率 → 布基礎長 / 束石数量
   ├─[軸部] 延床 × 材積率 [× 補正率] → 木材材積量
   ├─[屋根] 伏面積 × 勾配伸び率
   ├─[外壁] 周長 × 壁高 竏鈀 開口部面積
   ├─[内壁] 各室周長 × 天井高 竏鈀 開口部面積
   ├─[床]   各室面積（仕上材種別）
   ├─[天井] 各室面積（仕上材種別）
   ├─[建具] 調査に基づく個別数量
   ├─[造作] 調査に基づく個別数量
   ├─[樋]   1階床面積
   ├─[塗装] 延床面積
   ├─[設備] 調査に基づく個別数量
   │
   ├─ 直接工事費（上記の合計）
   ├─ 共通仮設費 = 直接工事費 × 3%
   ├─ 純工事費 = 直接工事費 + 共通仮設費
   ├─ 諸経費 = 純工事費 × 諸経費率
   └─ 推定再建築費 = 純工事費 + 諸経費
```
